# Backpack Exchange trading bot

Currently support <b>SOL-USDC & PYTH-USDC</b> pairs

## Installation

```shell
git clone https://github.com/0xmusashi/backpack-trading.git
cd backpack-trading
cp .env.example .env
```

Fill your API_KEY & API_SECRET to .env file


## Usage

### 1. SOL-USDC trading
```shell
node index.js
```

### 2. PYTH-USDC trading
```shell
node pyth.js
```
